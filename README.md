# CNI-landscape-positioning-explorer
A prototype exploration of CNI strategic positioning and theme alignment in comparison to similar organizations 

This multi-tab exploration demonstrates CNI's strategic positioning in the ecosystem of 19 comparable organizations. This resource can be paired with the (CNI Landscape of Comparable Organizations Explorer)[https://meghaninmotion.github.io/CNI-landscape-comparable-orgs/] which maps the dominant themes and commitments across 19 library and IT member organization mission and vision statements, across three tabs: a themes + organizations matrix and themes overview.

This exploraton was developed by pairing with Claude AI to develop html to represent the overlapping and distinct objectives of similar organizations to CNI. It is presented for exploration. It can be further developed with your feedback and suggestions. Please share those thoughts - and report errors - as pull requests!
